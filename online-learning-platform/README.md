# 🚀 練課室 SkillSync — 創新型線上學習與個教平台 Prototype 0

> **全台名師陣容 • 實戰陪跑個教**
> 本專案為結合近 10 年職業培訓機構資源轉化之次世代技能學習平台。

---

## 🌟 平台亮點與特色

1. **三階融合學習法 (Tri-Learning Model)**：
   - 📹 **錄播自學**：打底基礎理論與範例演練
   - 🧑‍🏫 **1-on-1 名師個教**：專屬 Code Review、Figma 設計稿精細批改
   - 🎥 **同步直播小班**：3~6 人小組專案攻堅與 Live 解惑
2. **多角色動態權限管理 (RBAC)**：
   - 👑 **Wen總監 (平台主管)**：全站最高權限、帳號密碼管理、創業完整規劃書
   - 🧑‍💼 **營運員工**：課程/導師/錄播單元編輯上架
   - 🎓 **消費者學員**：課程瀏覽、商城購買、視訊學習與作業繳交
3. **無縫試聽與模擬體驗**：
   - 內建智慧 AI 隨課陪讀助教
   - 內建 1 對 1 個教預約日曆與線上畫布/直播教室

## 🚀 快速開始 (Quick Start)

### 免安裝任何伺服器直接開啟：
1. 下載或 Clone 本專案。
2. 雙擊開啟 `index.html` 即可在瀏覽器開啟完整功能。

### 使用 Node.js 本地開發伺服器：
```bash
npm install
npm run dev
# 訪問 http://localhost:3000
```
