🎯 二、 市場痛點與三大創新差異化優勢

  1. 三階融合學習法 (Tri-Learning Model)：錄播自學 (打底) ➔ 1-on-1 講師專屬 Code / 作品精細批改 (攻堅) ➔ 3~6人小班直播 (實戰)。完課率高達 90%+。
  2. 資深名師高分潤誘因 (65%~75% 拆帳)：善用近 10 年培訓機構累積之 Top 10% 口碑名師，遠高於補習班固定時薪。
  3. 結果導向與副業/求職變現媒合：學員完成作品後納入平台人才庫，講師直接推薦商業案源或企業職缺。

💳 五、 金流系統串接與學員權限開通管理機制

雙軌自動化與後台管理雙重架構：平台支援「學員線上自主刷卡/ Pay 自動開通」與「總監/幹部後台手動開通」雙重權限管理，確保線上銷售與線下企業包班收費皆能無縫對接。

2. 正式營運第三方金流 (綠界 ECPay / 藍新 NewebPay) 串接 3 步驟 Roadmap

  1. 特店帳號申請與金流審核：向台灣綠界科技 (ECPay) 或藍新金流 (NewebPay) 遞交營業登記與商店資料，取得 MerchantID、HashKey 與 HashIV 密鑰。
  2. 前台 API 結帳 Form 轉跳串接：將前台結帳按鈕 (handleCheckout) 連接至金流網關，支援信用卡一次付清/分期、LINE Pay、街口支付與 ATM 虛擬帳號轉帳。
  3. 背景自動化開通 (ReturnURL / Webhook)：建立安全驗證伺服器接收金流扣款成功通知 (Callback Signal)，背景自動建立學員帳號、發送 Email 開通通知信與發票電子化開立。
